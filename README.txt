# ESIGELEC_Projet_S8

Bonjour à tous les étudiants de l'ESIG. Ceci est mon projet S8 portant sur la réalisation d'un "Harsh Sensor".

Le but du projet : réaliser un boitier permettant de mesurer dans quelles conditions un ouvrier est soumis afin de définir s'il a droit a des compensations financières.

Vous trouverez sur ce git : 
  - le code des capteurs et du datalogging sur la Rpi 3B+
  - le code de la partie serveur Rpi 3B+
  - le code LABVIEW permettant de recevoir les données sur votre PC et les traiter (analyser les critères de pénibilités)
  (Le code LABVIEW possède quelques bugs sur la partie graphique mais la partie permettant de définir si l'ouvrier à droit à une compensation est fonctionnelle)
  
Voila, bon courrage ;)
